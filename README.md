A WIP modern taskbar for XFCE
C++ - GTK3 - CSS - DOCK-LIKE - MINIMALISM

Docklike is now published in the ArchLinux User Repository (AUR) !

Installation instructions :
```
# For ArchLinux users :
yaourt -S xfce4-docklike-plugin-git

# On any other distro :
./autogen.sh
make
sudo make install
```

Building dependency :
libxfce4panel-2.0
libxfce4ui-2
gtk-3.0
cairo-1.16
libwnck-3.0
